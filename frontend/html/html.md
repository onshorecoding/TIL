## HTML이란
HTML(Hypertext Markup Language)는 프로그래밍 언어는 아니고, 웹페이지가 어떻게 구주화 되어있는지 브라우저로 알 수 있도록 하는 마크업 언어입니다.

HTML은 element로 구성되으며, 이에 대한 종류로는 Block-level elements와 Inline elements가 있습니다.


## HTML의 구조
- html: 전체 html 문서를 감싸는 태그이다. 하나만 존재해야하고, DOCTYPE을 제외한 다른 태그가 있으면 안 됩니다.

- head: hltml 문서에 대한 정보를 나타내는 정보입니다. 하나만 존재해야하고 html 바로 아래 있어야합니다.

- body: html 문서에서 실제적으로 보여지는 부분입니다. 하나만 존재해야하고 html 바로 아래 head 다음에 위치합니다.

- title: head 안에 들어가는 태그로 제목표시줄의 내용을 나타냅니다.

- meta: 문서에대한 설명을 표시합니다
